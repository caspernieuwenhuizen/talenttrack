# Resetting your password

If you forget your password, you can set a new one yourself — without leaving the academy's own branded screens.

## How it works

1. On the sign-in card, choose **Lost your password?**
2. Enter your **email or username** and choose **Send reset link**. You'll always see the same confirmation — "if that account exists, we've sent a reset link" — so no one can use the form to discover which emails have an account.
3. Open the email and follow the **reset link**. It lands on the academy's branded "Choose a new password" screen.
4. Enter your new password twice (at least 8 characters) and choose **Set new password**.
5. You're returned to the sign-in card with a confirmation, ready to sign in.

## Notes

- Reset links **expire** and can be used **once**. If yours has expired, the screen offers to send a new one.
- The whole flow stays on the academy's branded pages — the logo and academy name you see when signing in. The underlying security (link generation, expiry, and password storage) is handled by WordPress.
- If you didn't request a reset, you can ignore the email; your password stays the same.

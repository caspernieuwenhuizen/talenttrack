# Player rate cards

The **Player Rate Cards** page is where you dive deep into a single player's performance.

## Components

- **FIFA-style card** — overall rating, key attributes, visual identity
- **Headline numbers** — most recent, rolling last-5 average, all-time average, total evaluations
- **Main category breakdown** — radar chart of the latest eval across all main categories
- **Subcategory breakdown** — table of every subcategory the player has been rated on, with all-time averages
- **Trend chart** — line chart of overall rating over time

## Filters

- **Date range** — only evaluations in this window count toward averages and the trend chart
- **Evaluation type** — restrict to "Match" evals only, for example, to see game-time performance separately from training assessments

## Printing

The top-right print button renders a clean printable version — same data, no admin chrome. Use the browser's "Save as PDF" to export.

## Permissions

Coaches with `tt_view_reports` see rate cards for any player. Players see only their own via the frontend. The Read-Only Observer role can view rate cards but nothing else.

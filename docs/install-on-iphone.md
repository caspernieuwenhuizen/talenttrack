# Install TalentTrack on your iPhone

You can install TalentTrack as an app on your iPhone in three taps. The app opens fullscreen and can send you notifications about new tasks, evaluations, and goal updates.

## You need

- iPhone running iOS 16.4 or newer.
- Safari (the install button only appears in Safari — Chrome on iPhone uses Safari underneath but can't install apps).

## Steps

1. Open TalentTrack in Safari and sign in.
2. Tap the **Share** button at the bottom of the screen (the square with the arrow pointing up).
3. Scroll down and tap **Add to Home Screen**.
4. Tap **Add** in the top right.

A TalentTrack icon now sits on your home screen. Tap it like any other app.

## After install

- Open the app and accept **Allow Notifications** when prompted. Without this you won't see push notifications even if your account has them enabled.
- The first push notification you receive verifies the phone number on your TalentTrack account — no SMS code needed.

## Troubleshooting

- **No share button** — make sure you're in Safari, not in another browser. Chrome and Firefox on iPhone don't show the install option.
- **Notifications never arrive** — check Settings → Notifications → TalentTrack. The toggle must be on, and "Lock Screen", "Notification Centre", and "Banners" must each be enabled.
- **Reinstalling** — long-press the icon, choose **Remove App**, then repeat the install steps.

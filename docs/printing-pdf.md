# Printing and PDFs

TalentTrack can produce clean, print-ready views of player reports — useful for parent meetings, evaluation handouts or paper records.

## What you can print

Pages with a print icon in the top right:

- **Player rate cards** — the full card, radar and trend.
- **Evaluation detail** — one evaluation with all categories and notes.

## How to print

Click the print icon. A clean version of the page opens in a new window with no menus or sidebars, and the browser's print dialog pops up.

## How to save as a PDF

In the print dialog, choose **Save as PDF** as the destination. Every modern browser supports this.

Tips:

- Set page size to A4 (or Letter in the US).
- Set margins to **Default** or **Normal**.
- Landscape works better for rate cards because the radar and trend charts have more room.

When you're done, close the print window. Your main TalentTrack tab stays exactly where it was.

## Branding

The print layout uses your academy's primary colour and logo. If a printout looks plain, ask an admin to set the academy logo and primary colour in **Configuration → Branding**.

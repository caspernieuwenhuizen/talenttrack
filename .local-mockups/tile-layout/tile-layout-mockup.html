Tile layout — “icon + title on top line, description beneath”

Proposed as a new tile_appearance layout option (alongside the current compact / comfortable / spacious sizing). Every grid below uses the same standard 220 px column width, 10 px gap, 14 px padding, 8 px radius — only the inside arrangement changes.

Title clamps to 2 lines Icon in the new layout = 40 px (≈ two title rows) Description = muted, beneath, full width

A · Current layout (for comparison)

Icon is vertically centred beside the text block; title and description stack to the right of the icon. A long title competes with the icon for horizontal room.

B · New layout — icon + title share the top line, description spans beneath

The icon sits beside the title on the first line and is tall enough that the title can wrap to a second row next to it — so the tile never has to grow wider than standard. The description drops to the full tile width underneath, with room for up to three lines.

B · New layout — single column at ≤ 640 px (mobile)

Same arrangement; the grid collapses to one column. Touch target stays ≥ 48 px tall.
